# Gerenciador de Calorias
Site desenvolvido para controle de refeições, macro nutrientes e calorias

## Time:
- PO: Hugo Nomura
- SM: Pedro Satoru 
- DEVs: Vitor Vianna e Pedro Correia

## Sprint 1:

### Objetivo da Sprint 1
O foco desta sprint de duas semanas é desenvolver todo o escopo inicial do projeto. Isso inclui desde a gestão de perfis de usuários até a funcionalidade principal de processar refeições via texto utilizando integração com IA, garantindo que a base do produto (MVP) seja sólida.
